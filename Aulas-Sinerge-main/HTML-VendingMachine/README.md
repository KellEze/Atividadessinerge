# HTML-VendingMachine (Máquina de Vendas)

Conforme `HTML-VendingMachine/Instrucoes.md`.

## Como usar
- Abra `index.html` no navegador.

## Funcionalidades
- Cards de produtos (escolha do produto e destaque).
- Entrada de valor pago.
- Comparação com preço.
- Cálculo de troco e quebra em notas/moedas (10,5,2,1).
- Mensagens de sucesso/erro.
- Botão Limpar.
