# Portal Web da Lanchonete (Flask)

## Rotas
- `/` → página inicial
- `/conteudos` → conteúdos estudados
- `/produtos` → lista de produtos
- `/cardapio` → cardápio
- `/sobre` → sobre o projeto
- `/clientes` → clientes fictícios (desafio extra)

## Como executar
1. Instale Flask:
   - `pip install flask`
2. Execute:
   - `python app.py`
3. Abra:
   - `http://127.0.0.1:5000`
